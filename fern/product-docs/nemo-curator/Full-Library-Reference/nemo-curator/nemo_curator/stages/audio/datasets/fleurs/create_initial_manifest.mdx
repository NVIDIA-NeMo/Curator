Stage to create initial manifest for the FLEURS dataset.

Dataset link: https://huggingface.co/datasets/google/fleurs

Will download all files, extract them, and create a manifest file with the
"audio_filepath" and "text" fields.

**Parameters:**

<ParamField path="lang" type="str">
Language to be processed, identified by a combination of ISO 639-1 and ISO 3166-1 alpha-2 codes.
Examples are:

- ``"hy_am"`` for Armenian
- ``"ko_kr"`` for Korean
</ParamField>

<ParamField path="split" type="str">
Which dataset splits to process.
Options are:

- ``"test"``
- ``"train"``
- ``"dev"``
</ParamField>

<ParamField path="raw_data_dir" type="str">
Path to the folder where the data archive should be downloaded and extracted.
</ParamField>

**Returns:**

This srage generates an initial SpeechObject with the following fields:

&#123;
    "audio_filepath": &lt;path to the audio file&gt;,
    "text": &lt;transcription&gt;,
