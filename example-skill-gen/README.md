## Quick Test

### Option 1: Local Test (Any Platform)

```bash
cd example-skill-gen

# Use --simple flag for local testing without NeMo Curator
python run_pipeline.py --simple
```

### Option 2: Docker (Full NeMo Curator)

```bash
cd example-skill-gen

docker run --rm \
  -v $(pwd):/workspace \
  -w /workspace \
  python:3.11-slim \
  bash -c "pip install nemo-curator && python run_pipeline.py"
```

## Usage with Your Data

```bash
cd example-skill-gen

python -m nemo_curator.config.run \
  --config-path=. \
  --config-name=text_quick_pass \
  input_path=/path/to/your/data \
  output_path=/path/to/filtered/output
```

## Customization

### Change text field name

```bash
python -m nemo_curator.config.run \
  --config-path=. \
  --config-name=text_quick_pass \
  input_path=/data/raw \
  output_path=/data/filtered \
  text_field=content
```

### Adjust filter thresholds

Edit the YAML directly or override on command line:

```bash
# More permissive word count (keep shorter docs)
python -m nemo_curator.config.run \
  --config-path=. \
  --config-name=text_quick_pass \
  input_path=/data/raw \
  output_path=/data/filtered \
  'stages.1.filter_obj.min_words=20'
```
